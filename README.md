# BLE WiFi 配网项目

这是一个基于 ESP32 和微信小程序的蓝牙 WiFi 配网解决方案。

⚠️ **安全提示**：本项目使用明文传输WiFi配置，仅适用于内网环境或可信设备间通信。

## 项目结构

- `xn_blufi_esp32/` - ESP32 固件项目（ESP32-S3）
- `blufi_wechat/` - 微信小程序项目

## 主要特性

- ✅ 蓝牙配网（BluFi协议）
- ✅ 多WiFi配置存储（最多10个）
- ✅ WiFi自动连接和重连
- ✅ WiFi扫描功能
- ✅ 模块化三层架构
- ✅ 微信小程序完整UI
